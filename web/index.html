Search, visualize and analyze your log files and operational data

Logscape is a big-data analysis tool. Using your logs and other data, Logscape produces clear dashboards and provides real-time alerts. All data is indexed on your system, whether locally, or using a centralised index store. Once indexed you can search the entire stack for a single term, or use precise functions and filters. Logscape is deployed on your infrastructure and provides a straight forward web interface, with enterprise ready access controls.

See how it works.

Open Source vs Enterprise

Logscape now has an open source offering available on Github. This is offered in conjunction with the Logscape enterprise level offering. Our open source repository is a restricted offering with all LDAP and Active Directory integrations removed, leaving only the default Logscape user management. The Open Source repository also has no official support outside of the Logscape community forums, and the existing publicly available support material.
